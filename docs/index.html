About Neko

Neko is a classic desktop pet that follows your mouse cursor around the screen. Originally created by Masayuki Koba for X-Windows and ported to Windows 95/98 by David Harvey in 1998, this JavaScript version brings the beloved cat to modern web browsers.

Installation

Quick Start (Auto-start)

Add this single line to your HTML to automatically start Neko:

<script src="https://yourusername.github.io/nekojs/neko.js" data-autostart></script>

Manual Control

For more control, initialize Neko programmatically:

<script src="https://yourusername.github.io/nekojs/neko.js"></script>
<script>
    // Create and start Neko
    const neko = createNeko({
        speed: 10,                    // Movement speed
        behaviorMode: 0,              // 0 = chase mouse
        idleThreshold: 8              // Distance to stop moving
    });
</script>

Behavior Modes

Neko supports different behavior patterns:

  • 0 - Chase Mouse (default)
  • 1 - Run Away From Mouse
  • 2 - Run Around Randomly
  • 3 - Pace Around Screen
  • 4 - Run Around (chase invisible ball)

API Methods

const neko = createNeko();
neko.start();          // Start the animation loop
neko.stop();           // Stop the animation loop
neko.destroy();        // Remove from page
neko.behaviorMode = 2; // Change behavior
